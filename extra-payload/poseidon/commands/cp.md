## Summary
Copy a file from one location to another.
  
- Needs Admin: False  
- Version: 1  
- Author: @xorrior  

### Arguments

#### source

- Description: Source file to copy.  
- Required Value: True  
- Default Value: None  

#### destination

- Description: Source will copy to this location  
- Required Value: True  
- Default Value: None  

## Usage

```
cp -source test -destination test.bak
```
