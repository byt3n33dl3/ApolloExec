## Summary
Get information about mounted drives.

- Needs Admin: False  
- Version: 1  
- Author: @djhohnstein 

### Arguments

## Usage

```
drives
```

## MITRE ATT&CK Mapping

## Detailed Summary

This command use the os.Stat function in Golang to enumerate the `/mnt` and `/Volumes` directories. This command is only available for nix systems.
